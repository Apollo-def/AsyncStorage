# Tarefas Aula

Uma aplicação simples de gerenciamento de tarefas construída com React Native e Expo.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas

## Instalação

1. Instale as dependências: `npm install`
2. Inicie a aplicação: `npm start`

## Uso

- Executar no Android: `npm run android`
- Executar no iOS: `npm run ios`
- Executar na web: `npm run web`

## Estrutura do Projeto

- `App.js`: Componente principal da aplicação (contém TaskInput)
- `index.js`: Ponto de entrada
- `src/components/`: TaskInput, TaskItem, TaskList
- `src/hooks/`: Hook useTasks
- `src/styles/`: estilos globais

## Tecnologias

- React Native
- Expo
